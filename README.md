# ios-bullion

## Minimum Deployments
iOS 17.0

## Build With
- UIKit Programatically
- Clean architecture with MVVM
- Combine Framework
- Keychain
- UserDefaults
- CryptoKit
- UICollectionViewCompositionalLayout 
- CAGradientLayer

## Features
- Pagination (Scroll to bottom)
- Alert message
- Loading Indicator
- Custom Dialog
